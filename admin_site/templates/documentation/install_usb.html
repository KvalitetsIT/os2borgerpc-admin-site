Installation af BibOS via USB

Installation fra USB foregår på stort set samme måde som installation via DVD. Du skal blot bruge programmet tuxboot til at overføre ISO'et til USB-stick'en i stedet for at brænde en DVD.

Bemærk at USB-stick'en vil optræde som en harddisk på computeren under installationen, så vær sikker på at vælge den rigtige harddisk når du bliver spurgt hvor du vil installere.
